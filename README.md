# 森德 Zehnder 康舒安新风 Home Assistant 集成

这是一个用于 Home Assistant 的 HACS 自定义集成，面向森德 Zehnder 康舒安 CA-D635EC 新风换气机和 CA-H3-5S 控制模块。

集成通过本地 Modbus 轮询读取和控制设备，不依赖 Smart Life App 或云端服务。

## 兼容版本

- Home Assistant: `2025.3.0` 及之后版本
- 安装方式: HACS 自定义集成或手动复制 `custom_components`

## 支持设备

- 新风换气机型号: `CA-D635EC`
- 控制模块型号: `CA-H3-5S`
- 通讯接口: 控制模块 RS485 接口
- 协议: Modbus
- 可选传感器: PM2.5、CO2、TVOC/VOC

协议和说明书来源于当前仓库内 PDF:

- `CA-H3通讯协议-V1.0.pdf`
- `控制器CA-H3-5S-20250702.pdf`

## 安装

### 通过 HACS 安装

1. 在 HACS 中添加此仓库为自定义仓库。
2. 类型选择 `Integration`。
3. 安装 `Zehnder ComfoAir CN`。
4. 重启 Home Assistant。
5. 进入 **设置 -> 设备与服务 -> 添加集成**，搜索 `Zehnder ComfoAir CN`。

### 手动安装

将 `custom_components/zehnder_comfoair_cn` 复制到 Home Assistant 配置目录下的 `custom_components` 中，然后重启 Home Assistant。

## 配置页面

有配置页面。

在 Home Assistant 的集成添加流程中，会先选择连接方式:

- `TCP/IP RS485 网关`: 适用于网口转 RS485、串口服务器、Modbus TCP 网关等设备。
- `本地 USB-RS485 串口`: 适用于 USB-RS485 适配器直接插在 Home Assistant 主机上的场景。

### TCP/IP RS485 网关配置项

- IP 地址或主机名
- TCP 端口，默认 `502`
- Modbus 地址，默认 `1`
- 波特率，默认 `9600`
- 校验，默认无校验
- 数据位，默认 `8`
- 停止位，默认 `1`

注意: 如果使用 TCP/IP 转 RS485 网关，波特率、校验、数据位、停止位是网关 RS485 侧的参数，需要在网关设备中设置一致。Home Assistant 通过 IP 和 TCP 端口连接网关本身，通常不能直接修改网关的串口侧参数。

### 本地 USB-RS485 串口配置项

- 串口路径，例如 `/dev/ttyUSB0`、`/dev/ttyACM0`、`/dev/serial/by-id/...`
- Modbus 地址，默认 `1`
- 波特率，默认 `9600`
- 校验，默认无校验
- 数据位，默认 `8`
- 停止位，默认 `1`

### 控制器默认通讯参数

根据 `控制器CA-H3-5S-20250702.pdf` 中的用户参数设置:

- `U07` 通讯地址: 默认 `1`，范围 `0-247`
- `U08` 通讯波特率: 默认 `2`，对应 `9600`
  - `1`: `4800`
  - `2`: `9600`
  - `3`: `14400`
  - `4`: `19200`
  - `5`: `38400`
- `U09` 通讯校验方式: 默认 `0`，无校验
  - `0`: 无校验
  - `1`: 奇校验
  - `2`: 偶校验

## 实体

集成会创建以下实体:

- 风机: 开关机、低/中/高风速、运行模式
- 传感器: 室内温度、室内湿度、室外温度、PM2.5、CO2、VOC
- 二进制传感器: 滤网报警、防冻报警
- 开关: 增压继电器、混风继电器
- 按钮: 复位滤网报警

## 协议寄存器

### 输入寄存器

使用功能码 `04` 读取:

| 寄存器 | 名称 | 说明 |
| --- | --- | --- |
| `0x0000` | 室内温度 | `-50..400` 对应 `-5.0..40.0 C` |
| `0x0001` | 室内湿度 | `0..990` 对应 `0..100.0 %` |
| `0x0002` | 室外温度 | `-15..40 C` |
| `0x0003` | PM2.5 | `0..999 ug/m3` |
| `0x0004` | CO2 | `400..5000 ppm` |
| `0x0005` | VOC | `0` 优，`1` 良，`2` 轻度，`3` 中度，`4` 重度 |
| `0x0006` | 滤网报警 | `0` 正常，`1` 报警 |
| `0x0007` | 防冻报警 | `0` 正常，`1` 报警 |

### 保持寄存器

使用功能码 `03` 读取，使用功能码 `06` 写入:

| 寄存器 | 名称 | 取值 |
| --- | --- | --- |
| `0x0000` | 滤网报警复位 | 写入 `1` |
| `0x0001` | 模式切换 | `0` 自动，`1` 新风，`2` 无，`3` 睡眠，`4` 旁通，`5` 防冻 |
| `0x0002` | 风速切换 | `1` 低速，`2` 中速，`3` 高速 |
| `0x0003` | 开/关机 | `0` 关机，`1` 开机 |
| `0x0004` | 增压继电器 | `0` 关闭，`1` 开启 |
| `0x0005` | 混风继电器 | `0` 关闭，`1` 开启 |

## 备注

- 协议说明中写明，自动模式在带 PM2.5 传感器时有效。
- PM2.5、CO2、VOC 是选配传感器；如果实际设备未安装，对应实体可能不可用或数值无意义。
- 如果使用网络 RS485 网关，但设备无法连接，优先检查网关的串口侧参数、A/B 接线、控制器通讯地址和网关是否使用标准 Modbus TCP。
